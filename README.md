# Q&A diary EC Landing Page

`Q&A diary` という日記商品のための、静的なECランディングページです。

質問に答えるだけで日記を書ける商品の魅力を、やさしく親しみやすい文具テイストで紹介し、購入までスムーズにつなぐ構成になっています。

## 特徴

- ファーストビューで商品価値と購入導線を見せる構成
- 文具・紙ものらしい、やわらかい配色と手書き感のあるタイポグラフィ
- 商品らしさを出すための表紙風ビジュアル、付箋、インデックス表現
- 購入セクションに数量変更、ギフト設定、値札なしオプションを実装
- レビュー、仕様、FAQ まで含めた静的ECページ
- 軽いアニメーションとマウス追従エフェクトあり

## ファイル構成

- `index.html`
  ページ全体の構造。MV、購入エリア、特長、使い方、レビュー、FAQ を定義しています。
- `styles.css`
  デザイン全体のスタイル。配色、レイアウト、紙ものらしい装飾、レスポンシブ調整を管理しています。
- `script.js`
  スクロール時の表示アニメーション、数量ボタン、マウス追従の演出を管理しています。
- `assets/qa-diary.jpg`
  商品画像用アセット。

## 使い方

特別なビルドは不要です。`index.html` をブラウザで開くと表示できます。

ローカルサーバーで確認したい場合は、たとえば以下のように起動できます。

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000` を開いて確認してください。

## カスタマイズしやすい箇所

- 商品名や価格
  `index.html` のヒーローと購入セクションを編集
- 配色やフォント
  `styles.css` の `:root` と `font-family` を編集
- FAQ やレビュー文言
  `index.html` の各セクションを書き換え
- 数量やギフト設定の挙動
  `script.js` を編集

## 今後の拡張例

- 購入ボタンを実際のカートURLへ接続
- 数量やギフト設定に応じた合計金額の動的更新
- 商品画像ギャラリーやサムネイル切り替えの追加
- FAQ の開閉アニメーション強化
- 実在商品の情報に合わせた仕様・配送文言の更新

## 注意

このプロジェクトは現在、フロントエンドのみの静的実装です。

- 決済機能は未接続です
- 在庫や価格の動的管理はありません
- FAQ やレビューはデモ用の文言です
